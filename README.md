# My Backend Learning Projects 

This repository is a collection of small practice projects.
The main idea is to build mini-projects to learn and understand backend concepts step by step.


## Projects List

* [**FastAPI Basics**](./fastapi_basics) - Learning how to build a basic backend server and use CRUD operations (GET, POST, PUT, DELETE).
* [**API Testing**](./api_testing) - Learning QA Automation by writing integration tests for the API using `pytest` and `TestClient`.
* [**Security Basics**](./security_basics) - Learning core user authentication logic, password hashing with `bcrypt`, and data validation.

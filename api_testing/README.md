# API Testing with Pytest 

This mini-project focuses on **QA Automation** and Backend Testing. It contains automated integration tests for the shoe store API.

## What I learned here:
* Using `pytest` as a testing framework.
* Using FastAPI's `TestClient` to test the server without actually running it.
* **Positive & Negative Testing:** Checking if the server works correctly, and checking if it handles errors properly (e.g., shoe not found).
* **Integration Testing:** Testing a full lifecycle (Create -> Read -> Update -> Delete) in one automated flow.

## How to run the tests:
Open the terminal in the **root folder** of the repository and run:
`pytest`
